# Nova Link 动画控制

使用 scripts 目录下的脚本控制 Nova Link 桌面宠物的动画和表情。

## 使用方法

在 OpenClaw 中调用这些脚本来触发 Nova Link 的动画。

## 脚本列表

| 脚本 | 说明 |
|------|------|
| happy.bat | 开心动画 |
| sad.bat | 伤心动画 |
| surprised.bat | 惊讶动画 |
| angry.bat | 生气动画 |
| dance.bat | 跳舞动画 |
| greeting.bat | 打招呼动画 |
| thinking.bat | 思考动画 |
| sleeping.bat | 睡觉动画 |
| idle.bat | 待机动画 |

## 参数说明

动画持续时间默认为 2000ms，可根据需要修改脚本中的 duration 参数。
